📌 Project Overview

Dental treatments often have unclear or hidden pricing.
This project solves that by providing a simple interface where users can select treatments and instantly see estimated costs.

The goal of this project is to simulate a real-world dental clinic pricing system with a clear and user-friendly experience.

✨ Features

🦷 Treatment Price Estimation
Select different dental procedures and instantly view cost estimates.

📊 Dynamic Pricing Calculation
Automatically calculates total costs based on selected services.

🎨 Clean User Interface
Minimal and intuitive design for easy usability.

⚡ Fast Performance
Lightweight and optimized frontend.

📱 Responsive Design
Works smoothly on desktop, tablet, and mobile devices.
